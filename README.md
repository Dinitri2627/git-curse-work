# VPN Note System

Консольная система заметок для ИТ-инфраструктуры VPN-сервиса.

## Функциональность

- **Аутентификация** — вход/выход с хранением паролей в хешированном виде (BCrypt)
- **Заметки** — создание, просмотр, список, мягкое удаление
- **Мониторинг системы** — сбор статистики CPU, RAM, HDD
- **Логи безопасности** — запись всех событий аутентификации и операций с заметками
- **Автообновление** — проверка и загрузка обновлений
- **Справка** — встроенная Markdown-документация

## Технологии

- **Язык**: C# (.NET 11.0)
- **СУБД**: PostgreSQL (через Entity Framework Core + Npgsql)
- **ОС**: Windows 10 / 11
- **Тесты**: xUnit + Moq + EF Core InMemory

## Быстрый старт

1. Убедитесь, что PostgreSQL запущен
2. Настройте строку подключения в `appsettings.json`
3. Запустите приложение:

```bash
dotnet run --project VpnNoteSystem.App -- --help
```

## Команды

| Команда | Описание |
|---------|----------|
| `--login <user> <pass>` | Вход в систему |
| `--logout` | Выход |
| `--listusers` | Список всех пользователей (admin) |
| `--addNewNote "текст"` | Создать заметку |
| `--listNotes` | Список заметок |
| `--getNote <id>` | Просмотр заметки |
| `--deleteNote <id>` | Удалить заметку |
| `--stats` | Статистика CPU/RAM/HDD |
| `--securityLogs` | Логи безопасности |
| `--checkUpdate` | Проверка обновлений |
| `--update` | Установка обновления |
| `--help` | Справка |

## Тестирование

```bash
dotnet test VpnNoteSystem.Tests
```

### Тесты

1. **AuthTests** — проверка авторизации (логин, неверный пароль, неактивный пользователь, выход)
2. **DatabaseTests** — проверка подключения к БД (инициализация, CRUD для всех сущностей)
3. **UpdateTests** — проверка корректности обновлений (без сети, без авторизации, логирование)
4. **NoteTests** — проверка работы с заметками (создание, список, получение, удаление, изоляция пользователей)
