# VPN Note System — Справочное руководство

Консольная система для создания и управления заметками в ИТ-инфраструктуре VPN-сервиса. Поддерживает аутентификацию, мониторинг системы, журнал безопасности и разграничение прав (пользователь / администратор).

---

## Аутентификация

| Команда | Описание | Пример |
|---------|----------|--------|
| `--register <user> <pass>` | Регистрация нового пользователя | `--register user1 pass123` |
| `--login <user> <pass>` | Вход в систему | `--login user1 pass123` |
| `--logout` | Выход из системы | `--logout` |

### Команды администратора

| Команда | Описание | Пример |
|---------|----------|--------|
| `--listusers` | Список всех пользователей | `--listusers` |
| `--createadmin <user> <pass>` | Создать нового администратора | `--createadmin admin2 pass456` |
| `--deleteuser <id>` | Деактивировать пользователя | `--deleteuser 3` |
| `--usernotes <id>` | Просмотр заметок пользователя | `--usernotes 2` |

---

## Заметки

| Команда | Описание | Пример |
|---------|----------|--------|
| `--addnewnote "текст"` | Создать заметку | `--addnewnote "Настроить VPN-шлюз"` |
| `--listnotes` | Список моих заметок | `--listnotes` |
| `--getnote <id>` | Просмотр заметки | `--getnote 1` |
| `--deletenote <id>` | Удалить заметку | `--deletenote 1` |

---

## Система

| Команда | Описание |
|---------|----------|
| `--stats` | Статистика CPU, RAM, HDD |
| `--securitylogs` | Последние события безопасности |
| `--checkupdate` | Проверить обновления |
| `--update` | Установить обновление |

---

## Справка

| Команда | Описание |
|---------|----------|
| `--help` | Показать данное руководство |
| `exit` | Выйти из программы |

---

## Примеры

```bash
# Регистрация
VpnNoteSystem.exe --register user1 pass123

# Вход
VpnNoteSystem.exe --login user1 pass123

# Создание заметки
VpnNoteSystem.exe --addnewnote "Проверить балансировщик нагрузки"

# Просмотр заметок (своих)
VpnNoteSystem.exe --listnotes

# Статистика сервера
VpnNoteSystem.exe --stats

# Логи безопасности
VpnNoteSystem.exe --securitylogs
```

## Требования

- Windows 10 / 11
- PostgreSQL 14+ (или SQLite для тестирования)
- .NET 11.0 Runtime
