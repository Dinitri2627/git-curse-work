# VPN Note System - Справочное руководство

## Описание
Консольная система для создания и управления заметками в ИТ-инфраструктуре
VPN-сервиса. Поддерживает аутентификацию, мониторинг системы и журнал безопасности.

---

## Команды

### Аутентификация
```
--register <username> <password>
```
Регистрация нового пользователя. Пример: `--register user1 pass123`

```
--login <username> <password>
```
Вход в систему. Пример: `--login admin admin123`

```
--logout
```
Выход из системы.

```
--listusers
```
Просмотр всех зарегистрированных пользователей (только для администратора).

### Работа с заметками
```
--addNewNote "текст заметки"
```
Создать новую заметку. Пример: `--addNewNote "Настроить VPN-шлюз"`

```
--listNotes
```
Показать все заметки текущего пользователя.

```
--getNote <id>
```
Показать содержимое заметки. Пример: `--getNote 1`

```
--deleteNote <id>
```
Удалить заметку. Пример: `--deleteNote 1`

### Статистика системы
```
--stats
```
Показать загрузку CPU, RAM и HDD текущего устройства.

### Логи безопасности
```
--securityLogs
```
Показать последние события безопасности (входы, выходы, операции с заметками).

### Обновления
```
--checkUpdate
```
Проверить доступность новой версии.

```
--update
```
Загрузить и установить обновление.

### Справка
```
--help
```
Показать данное руководство.

---

## Примеры использования

```bash
# Регистрация нового пользователя
VpnNoteSystem.exe --register user1 pass123

# Вход в систему
VpnNoteSystem.exe --login admin admin123

# Создание заметки
VpnNoteSystem.exe --addNewNote "Проверить балансировщик нагрузки"

# Просмотр заметок
VpnNoteSystem.exe --listNotes

# Получение статистики сервера
VpnNoteSystem.exe --stats

# Просмотр логов безопасности
VpnNoteSystem.exe --securityLogs

# Проверка обновлений
VpnNoteSystem.exe --checkUpdate
```

## Требования
- Windows 10 / 11
- PostgreSQL 14+
- .NET 11.0 Runtime
